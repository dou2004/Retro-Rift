# 🕹️ Retro Rift *A Pure HTML/CSS/JavaScript Asteroids Arcade Game* [![Play Now](https://img.shields.io/badge/PLAY-DEMO-brightgreen?style=for-the-badge)](#) [![License](https://img.shields.io/badge/License-MIT-blue?style=for-the-badge)](LICENSE)

## 🌠 About **Retro Rift** is a modern reimagining of the 1979 arcade classic *Asteroids*, built entirely with vanilla web technologies. No frameworks, no external assets—just pure nostalgic chaos! ## 🚀 Features - **Physics-based movement** with dynamic thruster animations - **Destructible asteroids** that split into dangerous fragments - **Risk/reward hyperspace** (10% chance of disaster!) - **Parallax starfield** background (pure CSS) - **Progressive difficulty** with faster spawns over time - **Laser cooldown system** for strategic play ## 🎮 Controls | Key | Action | |-----|--------| | ↑ | Thrust | | ← → | Rotate | | Space | Fire Laser | | H | Hyperspace Jump | ## 🛠️ Installation 1.
